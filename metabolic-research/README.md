# Metabolic Research

A personal research project for secondary analysis (reanalysis) of public
metabolic datasets.

## Goal

Reanalyze published metabolic / omics datasets to test independent hypotheses —
downloading public data, processing it reproducibly, and producing tables and
figures.

## Layout

```
metabolic-research/
├── scripts/
│   ├── config.R              shared paths + settings (sourced by all scripts)
│   ├── 00_install_packages.R install dependencies (run once)
│   ├── 01_download_data.R    fetch public datasets into data/raw/
│   ├── 02_explore_data.R     inspect raw data structure
│   └── 03_analysis.R         core reanalysis → results/ + figures/
├── data/
│   ├── raw/                  downloaded source data (gitignored)
│   └── processed/            cleaned/derived data (gitignored)
├── figures/                  generated plots (gitignored)
└── results/                  generated tables (gitignored)
```

Run scripts in numbered order. Each one re-derives its inputs, so the project
is reproducible from a clean checkout.

## Getting started

```r
# from the metabolic-research/ directory
source("scripts/00_install_packages.R")
source("scripts/01_download_data.R")
source("scripts/02_explore_data.R")
source("scripts/03_analysis.R")
```

Or from the shell: `Rscript scripts/00_install_packages.R`

## Notes

- `data/`, `figures/`, and `results/` contents are gitignored — they are
  regenerated by the scripts. To version a specific processed file, adjust
  `.gitignore`.
- PDF output uses `cairo_pdf` (via `PDF_DEVICE` in `config.R`) to handle
  Unicode characters cleanly.

## Moving this to its own repository

This folder is self-contained. To make it a standalone repo on your machine:

```sh
cp -r metabolic-research ~/metabolic-research
cd ~/metabolic-research
git init
git add .
git commit -m "Initial commit: metabolic dataset reanalysis scaffold"
```
